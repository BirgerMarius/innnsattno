# innsatt.no Project Rules

Project name: innsatt.no

Framework: Laravel

Rules:

- Preserve the existing architecture.
- Use the shared layout at `resources/views/layouts/app.blade.php`.
- Use the shared header partial at `resources/views/partials/header.blade.php`.
- Use the shared footer partial at `resources/views/partials/footer.blade.php`.
- Use global custom CSS at `public/css/custom/app.css`.
- Make small incremental changes.
- Complete one logical task at a time.
- Do not deploy without user approval.
- Migrate older standalone modules only in separate Work Packages.
